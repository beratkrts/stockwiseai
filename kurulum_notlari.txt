Kurulum Notlari

1) WSL'de Ubuntu 22.04 kuruldu.
   Komutlar:
   - wsl --install -d Ubuntu-22.04

2) Ubuntu 22.04 icinde Docker Engine yuklendi.
   Komutlar:
   - sudo apt update
   - sudo apt install -y ca-certificates curl gnupg lsb-release
   - sudo install -m 0755 -d /etc/apt/keyrings
   - curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
   - echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
   - sudo apt update
   - sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
   - sudo usermod -aG docker $USER

3) Docker Engine icinde PostgreSQL 16 kuruldu.
   Komutlar:
   - docker pull postgres:16
   - docker run --name postgres16 -e POSTGRES_PASSWORD=YOUR_PASSWORD -p 5432:5432 -d postgres:16

Windows'ta servis gibi calistirma (NSSM):
1) NSSM indir ve nssm.exe'yi bir klasore koy (ornek: C:\Tools\nssm\nssm.exe).
2) Servis olustur:
   - C:\Tools\nssm\nssm.exe install tkis_raw_sync "C:\Users\BERAT.KARATASOGLU\Desktop\tkis_stockwise\.venv\Scripts\python.exe" "etl\raw_sync.py"
3) Calisma dizinini ayarla (kritik):
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppDirectory "C:\Users\BERAT.KARATASOGLU\Desktop\tkis_stockwise"
4) (Opsiyonel) Log ayarla:
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppStdout "C:\Users\BERAT.KARATASOGLU\Desktop\tkis_stockwise\logs\raw_sync.out.log"
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppStderr "C:\Users\BERAT.KARATASOGLU\Desktop\tkis_stockwise\logs\raw_sync.err.log"
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppRotateFiles 1
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppRotateBytes 10485760
5) (Opsiyonel) Restart davranisi:
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppRestartDelay 5000
   - C:\Tools\nssm\nssm.exe set tkis_raw_sync AppExit Default Restart
6) Baslat/durdur:
   - C:\Tools\nssm\nssm.exe start tkis_raw_sync
   - C:\Tools\nssm\nssm.exe stop tkis_raw_sync
7) Servis kaldirma:
   - C:\Tools\nssm\nssm.exe remove tkis_raw_sync confirm
